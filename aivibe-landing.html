Мощный набор технологий, работающих вместе как единая система

📡
LiDAR Сканирование
RoomPlan 2 создаёт точную 3D-модель комнаты с точностью ±3мм. Поддержка всех LiDAR-устройств iPhone. Автоматический экспорт в USDZ.
RoomPlan 2 USDZ ARKit
🤖
Российский AI
YandexGPT 5 Pro анализирует интерьер. GigaChat Ultra как резерв. Core ML оффлайн — работает без интернета. Triplex Fallback с Circuit Breaker.
YandexGPT 5 GigaChat Core ML
🪄
AR Дизайнер
RealityKit 4 и RealityView. Расставляй мебель через камеру — drag & drop в реальном пространстве. Undo/Redo до 20 действий. Lighting estimation.
RealityKit 4 RealityView SwiftUI
🛒
Маркетплейсы РФ
Товары с Wildberries, Ozon и СберМегаМаркет прямо в приложении. AI подбирает мебель под ваш стиль и бюджет с прямыми ссылками.
Wildberries Ozon СберМегаМаркет
📵
Оффлайн-режим
Core ML работает полностью без сети. Базовые рекомендации по стилю и цветам доступны даже в подвале. Lazy загрузка модели при первом обращении.
Core ML Offline On-device
🗂
Портфолио
Сохраняй AR-проекты с публичными ссылками. Yandex Object Storage для USDZ-файлов. Поделись с дизайнером или покажи строителям прямо с телефона.
Yandex Storage Share AR Links

Triplex Fallback.
Всегда на связи.

Если один провайдер недоступен — система автоматически переключается на следующий. Без задержек, без ошибок для пользователя.

YandexGPT 5 Pro
Yandex Cloud Foundation Models.
Анализ фото, рекомендации стиля, подбор мебели.
⚡ timeout 30с
→
02 / РЕЗЕРВ
GigaChat Ultra
Сбербанк AI. OAuth через backend-прокси.
Самоподписанный сертификат обработан.
⚡ timeout 30с
→
03 / ОФФЛАЙН
Core ML
On-device модель. Работает без сети.
Template matching для базовых запросов.
📵 offline only
⚡

Circuit Breaker Protection

После 3 ошибок подряд провайдер автоматически пропускается на 5 минут. Health check каждые 60 секунд восстанавливает доступность. Реализован как Swift 6 actor — потокобезопасно.

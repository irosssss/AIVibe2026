### ✅ Рекомендации по исправлению (TCA 1.19+ / Swift 6)

1. **Инициализация Store:**
   ```swift
   @StateObject private var store: StoreOf<RoomScanFeature>
   
   init() {
       _store = StateObject(wrappedValue: Store(initialState: .init()) {
           RoomScanFeature()
       })
   }
   ```
2. **Современное наблюдение за состоянием:**
   ```swift
   WithPerceptionTracking {
       if store.status == .idle { ... }
   }
   ```
   *(Если `State` помечен `@Observable`, то `WithPerceptionTracking` не обязателен, но рекомендуется для контроллируемого наблюдения.)*
3. **Защита от двойного скана:**
   ```swift
   Button("Start Scan") {
       viewStore.send(.startScan)
   }
   .disabled(store.status == .scanning)
   ```
4. **Условная компиляция RoomPlan:**
   ```swift
   #if canImport(RoomPlan)
   // код AR
   #else
   Text("LiDAR не поддерживается на этом устройстве")
   #endif
   ```
5. **Info.plist права:** Убедитесь, что добавлены ключи `Privacy - Camera Usage Description` и `Privacy - Local AR Usage Description`.
